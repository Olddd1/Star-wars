Игра "Звёздные войны" это минималистичное перерождение игры "Space Invaders".

При создании проекта использовалась исключительно библиотека pygame.

На данном этапе разработки игры присутствуют данные функции:
    - Увеличение сложности с каждым уровнем
    - Система подсчёта очков в зависимости от уровня
    - Сохранение рекорда в реальном времени

Основные классы:
    - Ufo (отвечает за поведения отдельного пришельца)
    - Bullet (отвечает за поведение пули)
    - Gun (отвечает за поведение пушки)

Основные функции:
    - screen_update (отвечает за отрисовку всех элементов экрана)
    - restart (отвечает за перезапуск игры при проигрыше)
    - load_image (отвечает за подгрузку изображений)

 Доработки в следующих версиях игры:
    - Создание защиты для игрока
    - Разнообразить пришельцев
    - Создание новых режимов
    - Улучшение графики
